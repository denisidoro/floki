# Floki

An EDN/JSON browser for the terminal

<img src="https://user-images.githubusercontent.com/3226564/50401816-89bf5b00-0778-11e9-9e98-8208e2bebd3e.gif"
     alt="Demo"
     width=520 />

### Installation

```sh
npm install -g floki
```

### Usage

```sh
echo '{"a": 42}' | floki
```

Then, simply use `HJKL` or the arrow keys for movement.
